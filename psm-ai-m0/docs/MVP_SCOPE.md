# MVP Scope

## In scope
1. Upload property documents.
2. Bob Analysis.
3. Territory Dashboard.
4. Opportunity Score.
5. Generate PSM Report.

## Demo case
Colina Condesa, using the uploaded masterplan, DEM, Bob portrait, and wellness/territory image.

## Out of scope for M0
Authentication, billing, multi-tenant storage, live GIS processing, consultant marketplace, and generalized municipal data integrations.
