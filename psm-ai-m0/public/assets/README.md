# Bob Asset Library

Bob is the AI Territory Inspector of PSM AI.

Folder structure

master/
- Canonical PNG poses.

expressions/
- Emotional states.

animations/
- MP4 animations.

audio/
- Voice and sound effects.

scenarios.json
- Maps application screens to Bob's animation and expression.

Design principles

Bob is not a chatbot.

Bob behaves like a professional territory inspector.

Every movement should feel calm, confident and intentional.

Never cartoonish.

Never exaggerated.

Animations should support the story, not distract from it.
