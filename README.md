# BOB – AI Territory Inspector

## Overview

BOB is an AI Territory Inspector that reads surveys, terrain, water systems, and environmental evidence to reveal what a piece of land wants to become before design begins.

Built for the OpenAI Build with Codex Hackathon using GPT-5.6 and Codex.

## Features

- AI Territory Inspection
- Interactive Territory Dashboard
- Opportunity Score
- PSM Report Generation
- Case Study: Colina Condesa
